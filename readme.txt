Перед началом написания проекта, если не установлены, то установить: 

	1. node.js с офф.сайта,
	2. установить git bash с офф.сайта,
	3. установить gulp глобально

В каждом новом проекте перед началом работы в терминале bash:
	
	1. npm i

Режимы работы и команды:

	1. Development (режим разработчика) - npm run dev
	2. Build (режим деплоя)             - npm run build
	3. Создание архива zip              - npm run zip
